﻿Од текстот: 

"{{{query}}}"

извлечи го временскиот период од-до. Ако во текстот е пронајден само почетен датум, користи го денешниот датум {{date_now}} како краен датум. 
Ако не можеш да најдеш конкретен период, но има описи како „од вчера“, „за вчера“, „сношти“, „неделава“, „претходната недела“, „тековниот месец“, „од почеток на годината“, 
„3 квартал лани“ и слични вариации, процени го периодот врз база на денешниот датум {{date_now}}.
Ако од текстот нема никаков временски опис или не можи да се процени периодот, користи предефиниран период од претходните 5 дена до денешниот датум.

**Примери:**

1. Текст: „Во периодот од 01.03.2024 до 15.03.2024 беа забележани повеќе пријави.“  
Анализа: Има јасно зададени два датуми. 
Одговор: 
```json
{
  "dateFrom": "01.03.2024",
  "dateTo": "15.03.2024"
}
```

2. Текст: „Истражувањето започна на 10.04.2025 и сè уште трае.“  
Анализа: Само почетен датум е пронајде, тогаш се користи денешениот датум за крај.  
Одговор:
```json
{
  "dateFrom": "10.04.2025",
  "dateTo": "30.04.2025"
}
```

3. Текст: „Излистај ги сите проблеми кои се пронајдени вчера.“  
Анализа: „вчера“ = 1 ден пред денешниот. Ако денешениот ден е 24.04.2025.
Одговор:  
```json
{
  "dateFrom": "23.04.2025",
  "dateTo": "23.04.2025"
}
```

4. Текст: „Неделава имавме зголемување на активности.“  
Анализа: „неделава“ = од понеделник до денес (ако денес е 01.04.2025 → четврток).  
Одговор:
```json
{
  "dateFrom": "31.03.2025",
  "dateTo": "01.04.2025"
}
```

5. Текст: „Во текот на претходната недела имаше проблеми со системот.“  
Анализа:Денешниот датум се користи да се идентификува преходната недела. Ако денешниот ден е 17.04.2025 - Претходната недела е 07.04.2025 – 13.04.2025. 
Одговор: 
```json
{
  "dateFrom": "14.04.2025",
  "dateTo": "20.04.2025"
}
```

6. Текст: „Тековниот месец добивме голем број барања.“  
Анализа: Од денешниот ден се заклчува тековен месец, пример (јануари), до денес.  
Одговор:
```json
{
  "dateFrom": "01.01.2025",
  "dateTo": "01.01.2025"
}
```

7. Текст: „Од почетокот на годината имаме зголемен обем на работа.“  
Одговор:
Анализа: Денешниот ден го земаме како краен датум и како индикатор за почеток на која година се работи. Ако денешниот датум е 23.04.1076 значи се работи за почеток на 1076.  
```json
{
  "dateFrom": "01.01.1076",
  "dateTo": "23.04.1076"
}
```

8. Текст: „Во 2 квартал лани имавме добри резултати.“  
Анализа: 2 квартал = април–јуни, лани = значи претходната година од годината на денешниот датум (пример денес е 01.01.2000).  
Одговор:
```json
{
  "dateFrom": "01.04.1999",
  "dateTo": "30.06.1099"
}
```

9. Текст: „Во зимата 2023 имаше значителни влијанија.“  
Анализа:Денешниот датум не игра улога, Зима = декември 2023 + јануари/февруари 2024.
Одговор:  
```json
{
  "dateFrom": "01.12.2023",
  "dateTo": "28.02.2024"
}
```

10. Текст: „Целата 2022 година беше критична за развојот.“  
Анализа: Денешниот датум не игра улога, се поставуваат границите од почеток до крај на 2022 година. 
Одговор:  
```json
{
  "dateFrom": "01.01.2022",
  "dateTo": "31.12.2022"
}

11. Текст: „Израчунај број на сати за понеделникот.“  
Анализа: Денешниот датум, пример 31.12.2025 среда, ќе ни покажи која е тековната недела и од таму датумот на понеделникот = 29.12.2025. Во случајот датумот на понеделникот ќе биди и почеток на временскиот опсег и крај.
Одговор:  
```json
{
  "dateFrom": "29.12.2025",
  "dateTo": "29.12.2025"
}
```"
